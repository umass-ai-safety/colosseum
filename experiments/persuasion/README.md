# Persuasion Experiments

This directory combines the former `persuasion_collusion` and `persuasion_hospital` experiment trees.

- `collusion/`: JiraTicket persuasion-collusion experiment.
- `hospital/`: HospitalEnvironment persuasion-collusion experiment.

## Run commands

```bash
python experiments/persuasion/collusion/run.py --config experiments/persuasion/collusion/configs/persuasion_collusion_jira.yaml
python experiments/persuasion/hospital/run.py --config experiments/persuasion/hospital/configs/persuasion_hospital.yaml
```
